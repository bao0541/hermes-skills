# 🤖 加密货币AI盯盘交易系统 v2.0

## 概述

AI驱动的主动盯盘交易系统，不是死板跑策略机器人。每30分钟定时触发AI分析，遇到波动自动加密检查频次。模拟盘验证稳定后再切真盘。

**核心资产：ETH/USDT**（可选BTC/USDT双币）

**v2.0 新增优化：** 动态止盈止损（按波动率）、分批止盈+移动止损、熔断机制

## 架构

```
cron 每30分钟 → 触发AI分析
       │
       ▼
crypto_agent.py → 输出市场+账户数据

       ▼

       │
       ▼
```

## AI决策逻辑

每次触发，AI分析以下维度：

1. **趋势方向** — 多时间框架对齐（15m/1h/4h）
2. **SuperTrend指标** — 趋势跟踪信号（ATR=10, 倍数=3），所有周期都向上才是强信号
3. **RSI位置** — 超买/超卖/中性
4. **MACD信号** — 金叉/死叉/柱线变化
5. **布林带** — 突破上下轨/带宽收窄扩张
6. **成交量** — 异常放量/缩量
7. **波动率** — ATR、日均波动幅度（决定动态TP/SL）
8. **持仓状态** — 浮盈浮亏、保证金占用、连败计数
9. **市场情绪** — 综合评分

### 🎯 动态止盈规则

```
日均波动率 < 3%（低波动）:
  第一止盈 +1.5% → 平一半 → 保本止损 → 挂第二止盈 +3%

日均波动率 3-5%（正常）:
  第一止盈 +2%   → 平一半 → 保本止损 → 挂第二止盈 +4%

日均波动率 > 5%（高波动）:
  第一止盈 +3%   → 平一半 → 保本止损 → 挂第二止盈 +6%
```

做空同理，方向向下。

> 设计思路：波动小的时候赚快钱（1.5%就走一半），波动大的时候让利润跑（3%才走一半）。保本止损确保另一半永远不会亏钱。

### 动态止盈止损规则（🅱）

根据波动率等级动态调整：

| 波动率等级 | 日均波动 | 止盈目标 | 止损目标 | 说明 |
|:---------:|:--------:|:-------:|:-------:|:----|
| very_low / low | <3% | **2%** | **2%** | 震荡市，窄幅目标 |
| normal | 3-5% | **3%** | **2%** | 正常波动，追求合理收益 |
| elevated | 5-8% | **4%** | **1.8%** | 波动增大，加大止盈空间，收紧止损 |
| high / extreme | >8% | **5%** | **1.5%** | 高波动，大止盈小止损防回撤 |

波动率数据在 `ai_summary.volatility_level` 和 `ai_summary.volatility_daily_pct`。

### 分批止盈规则（🅱+）

```
做多示例：
开仓价 P0
第一目标 T1 = P0 × (1 + 止盈%)        ← 到此处卖一半
第二目标 T2 = P0 × (1 + 止盈% × 2)     ← 另一半推移动止损

移动止损：
- 卖出一半后 → 余仓止损推到开仓价（保本）
- 价格涨到 T2 → 止损推到 T1（锁住第一目标利润）
- 从 T2 回撤到 T1 → 触发移动止损，平剩余仓位

做空反向：
第一目标 = P0 × (1 - 止盈%)
第二目标 = P0 × (1 - 止盈% × 2)
```

### 翻倍/熔断机制（🅲）

```
连败计数规则：
- 止损一次 → 连败+1，下次翻倍开仓（0.1→0.2→0.4）
- 盈利一次 → 连败重置为0，回到0.1初始仓位
- 连败≥3次 → 强制暂停24小时，连败重置

总熔断：
- 单日最大亏损超过初始资金15%（150U）→ 强制平仓，暂停至次日
```

### 决策选项

| 动作 | 条件举例 |
|------|---------|
| 开仓买入 | SuperTrend翻转+日线方向一致+波动率匹配 |
| 分批止盈（卖一半） | 价格触及第一目标T1 |
| 移动止损 | 价格触及T1后推保本，触及T2后推T1 |
| 止损退出 | 价格触及动态止损线 |
| 马丁格尔补仓 | 止损后连败<3，下次信号翻倍开仓 |
| 暂停交易 | 连败3次/单日亏损超15%/波动极端 |
| 空仓等待 | 无明显机会/方向不明确 |

> **v1.0 → v2.0 升级内容：**
> - 固定2%止盈止损 → 按波动率动态调整
> - 一次性全仓止盈 → 分批止盈+移动止损
> - 简单翻倍限制 → 熔断机制（24小时暂停+总亏损熔断）
> - 单向做多 → 双向交易（多空都做）

## 波动密集检测规则

当以下任一条件触发，自动排入10分钟密集检查：
- 最近1根15分钟K线涨跌幅 > 1.5%
- 波动率等级为 "high" 或 "extreme"
- 价格突破布林带上下轨
- 15分钟MACD柱线突增

## 切换真盘

1. 注册币安，获取API Key（仅开交易权限）
2. 安装 `ccxt` 配置真实API密钥
3. 修改 `crypto_account.py` 添加真盘执行器（继承模拟接口）
4. 冲U到账户
5. 关闭模拟盘，开启真盘cron
